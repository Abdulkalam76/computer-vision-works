# Image Geometric Transformations using OpenCV

This project demonstrates basic geometric transformations on images using OpenCV and Python.

## Transformations Included
- Translation
- Rotation
- Scaling
- Shearing
- Affine Transformation
- Perspective Transformation

## Requirements
- Python 3.x
- OpenCV
- NumPy
- Matplotlib

## How to Run
```bash
pip install -r requirements.txt
python geometric_transformations.py
